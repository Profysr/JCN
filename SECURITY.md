# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, report it privately via:
- Email: bilalahmad072@proton.me
- Or use GitHub's private vulnerability reporting: go to the "Security" tab → "Report a vulnerability"

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We aim to respond within 48 hours and will keep you updated as we investigate and patch the issue. Once resolved, we'll credit you in the release notes (unless you prefer to stay anonymous).
